#### 快捷键

##### 普通快捷键

| Hotkeys        | Description |
| -------------- | ----------- |
| cmd + ctrl + 1 | outline大纲 |
| cmd + ctrl + 3 | Files列表   |
|                |             |



##### table操作快捷键

| Hotkeys              | Description                   |
| -------------------- | ----------------------------- |
| alt + cmd + t        | 在当前位置插入table           |
| cmd + enter          | 插入一行                      |
| cmd + shift + delete | 删除当前行                    |
| shift + enter        | 当前单元格中加入<br \/>换行符 |
