#### 快捷键



table操作快捷键

| 快捷键           | 描述                          |
| ---------------- | ----------------------------- |
| cmd+t            | 在当前位置插入table           |
| cmd+enter        | 插入一行                      |
| cmd+shift+delete | 删除当前行                    |
| shift+enter      | 当前单元格中加入<br \/>换行符 |
