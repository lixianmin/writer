#### 快捷键


| Hotkeys       | Description            |
| ------------- | ---------------------- |
| F1            | 显示文档               |
| alt + F7      | find usages            |
| alt + F12     | terminal               |
| alt + cmd + b | 转到实现               |
| alt + cmd + L | 格式化                 |
| cmd + [       | 回退到上一次编辑的位置 |
| cmd + 1       | Explorer               |
| cmd + 7       | Structure              |
| cmd + o       | open class             |
| shift + F6    | rename                 |
| double shift  | search anywhere        |

#### 设置

1. Editor -> General -> Auto Import，设置自动导入
2. Tools -> File Watchers，点击+，加入go fmt，设置文件保存时自动格式化
