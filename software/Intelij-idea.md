#### 快捷键


| Hotkeys   | Description         |
| ------------- | ------------------- |
| F1 | 显示文档 |
| alt + enter   | import 缺失的类声明 |
| alt + F7      | find usages         |
| alt + F12     | terminal            |
| alt + cmd + b | 转到实现            |
| alt + cmd + L | 格式化              |
| cmd + [ | 回退到上一次编辑的位置 |
| cmd + 1   | Explorer            |
| cmd + 7   | Structure           |
| cmd + o       | open class ，长按cmd**连点两次o可以勾选include non-project items** |
| ctrl + r | 运行程序 |
| shift + F6 | rename              |
| double shift  | search anywhere     |
