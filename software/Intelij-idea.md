#### 快捷键


| Hotkeys   | Description         |
| ------------- | ------------------- |
| alt + enter   | import 缺失的类声明 |
| alt + F7      | find usages         |
| alt + F12     | terminal            |
| alt + cmd + b | 转到实现            |
| alt + cmd + L | 格式化              |
| cmd + 1   | Explorer            |
| cmd + 7   | Structure           |
| cmd + o       | open class          |
| shift + F6 | rename              |
| double shift  | search anywhere     |
