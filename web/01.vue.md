#### 1 npm命令



```shell
npm init vite	# 初始化工程，需要使用自己的vpn(公司的代理不行)，否则无法执行

npm ls				# 打印安装的node模块
npm run dev		# 测试
npm run build # 打包； 如果需要运行的话，则需要启动一个http-server测试，比如brew install http-server
```
