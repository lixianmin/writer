



----

top打开后的控制按键：

| 按键 | 详解                      |
| ---- | ------------------------- |
| c    | 显示启动的command line    |
| e    | summary/task memory scale |
| P    | 按CPU使用率排序           |
| M    | 按内存占用率排序          |
| 1    | 显示CPU个数               |



```shell

# batch打印
top -b | grep risk-monitor

# 查看某一个pid的线程运行情况
top -Hp pid 


```
