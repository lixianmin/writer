#### 0x01 netstat

linux下建议使用netstat命令

|      |                                                  |
| ---- | ------------------------------------------------ |
| -a   | all，显示所有选项                                |
| -e   | 显示扩展信息，例如uid                            |
| -n   | number，拒绝显示别名，能显示数字的全部转化成数字 |
| -p   | program，显示建立相关链接的程序名                |
| -r   | route，显示路由信息                              |
| -s   | state，按各个协议进行统计                        |
| -t   | tcp，仅显示tcp相关选项                           |
|      |                                                  |
|      |                                                  |

#### 0x02 lsof

mac下建议使用lsof命令



|      |                                                       |
| ---- | ----------------------------------------------------- |
| -n   | 不显示主机名，不加该参数似乎意味着只打印新加入的进程  |
| -P   | 不显示端口俗称， *:mysql --> *:3306                   |
|      |                                                       |
| sudo | 如果不加，则只显示当前用户的，比如无法显示mysql的端口 |



```shell

# i一定要放到最后
sudo lsof -nP -i

# 打印listen的那h些进程
sudo lsof -nP -itcp -stcp:listen
```
