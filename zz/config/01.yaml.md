```yaml
# 1. 大小写敏感
# 2. ':'代表对象, '-'代表数组

 - Ruby
 - Perl
 - Python
 
# yaml使用缩进(只能用space, 不能用tab)表示层级关系. 缩进的空格数目不重要, 只要相同层级的元素左侧对齐即可
websites:
 YAML: yaml.org 
 Ruby: ruby-lang.org 
 Python: python.org 
 Perl: use.perl.org 

```
